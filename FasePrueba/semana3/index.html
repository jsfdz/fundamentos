Fundamentos del Desarrollo Web

HTML

👉🏻(del inglés HyperText Markup Language) no es un lenguaje de programación. Es un lenguaje de marcado que le dice a los navegadores web cómo estructurar las páginas web que estás visitando. Puede ser tan complejo o tan simple como desee el desarrollador web.

Una etiqueta es una palabra clave que nos permite definir un elemento en un archivo HTML.

Ejemplo de etiqueta:

            <tipo_de_elemento>

La mayoria de las etiquetas de HTML tienen una <etiqueta_de_apertura> que abre un elemento y una </etiqueta_de_cierre> que cierra un elemento.

Un Elemento es un tipo de componente que podemos utilizar para encerrar, delimitar o marcar diferentes partes del contenido para hacer que aparezcan de una cierta manera, o actúen de determinada forma en el HTML.

          <elemento> contenido </elemento>

Por ejemplo:

            <p> esto es un elemento de tipo párrafo </p>

🚀 Estructura Básica de HTML

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Titulo</title>
  </head>
  <body>
  </body>
</html>
          <!DOCTYPE html>

Declaración del tipo de documento es una instrucción especial que va al inicio de nuestro documento HTML y que permite al navegador entender qué versión de HTML estamos utilizando.

            <html lang="es">

Representa la raíz de un documento HTML (root element). El resto de elementos descienden de este elemento. etiqueta

El atributo lang lo usamos para definir el lenguaje en que están escritos los elementos, o el lenguaje en el cual deberían de estar escritos.

🤓 Los atributos de HTML son un modificador de un tipo de elemento de HTML, es decir, son palabras especiales utilizadas dentro de la etiqueta de apertura, para controlar el comportamiento del elemento. 

                <head>

Provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos. No es visible en la página. etiqueta

            <meta charset="UTF-8">

El atributo charset especifíca el tipo de caracteres que se utilizan en el documento. atributo

          <meta name="viewport" content="width=device-width, initial-scale=1.0">

El atributo viewport especifíca el tamaño de la ventana del navegador. Importante para hacer nuestras páginas responsivas. atributo

            <title>Título</title>

El atributo title especifíca el título del documento. atributo

Podemos agregar etiquetas adicionales como <meta name="description" content="Descripción del documento"> con el atributo description que especifíca una descripción del documento

o <link rel="icon" href="favicon.ico"> para mostrar el icono del documento en la pestaña del navegador.

Por último el elemento body representa el cuerpo del documento. etiqueta

            <body>

👉🏻 Elementos en bloque o en línea

Los elementos en HTML usualmente son elementos "en bloque" o elementos "en línea".

Por ejemplo:

  • div:
    Etiqueta para contenido en bloque
  • span: Etiqueta para contenido en línea

Un elemento en bloque ocupa todo el espacio de su elemento padre (contenedor), creando así un "bloque"

Un elemento en línea ocupa sólo el espacio delimitado por las etiquetas que definen el elemento en línea.

Hay un par de diferencias clave entre los elementos en bloque y elementos en línea:

  • Formateo
  • Modelo de contenido

Por ejemplo:

Elementos en bloque

  • De forma predeterminada, los elementos de bloque comienzan en una nueva línea, pero los elementos en línea pueden comenzar en cualquier parte de una línea.
  • En general, los elementos en bloque pueden contener elementos en línea y otros elementos en bloque. Inherente a esta distinción estructural es la idea de que los elementos en bloque crean estructuras "más grandes" que los elementos en línea.

Ver listado de elementos en bloque

Elementos en línea

  • De forma predeterminada, los elementos en línea no comienzan con la nueva línea.
  • En general, los elementos en línea pueden contener únicamente los datos y otros elementos en línea.

Ver listado de elementos en línea

👉🏻 Saltos

También podemos utilizar saltos de línea para separar elementos de texto así como saltos de contenido

Por ejemplo:

  • br: Etiqueta para dar un salto de linea
  • hr: Etiqueta para dar un salto de contenido (agrega un línea)

Algunos elementos consisten solo en una etiqueta única, que se utiliza generalmente para insertar/incrustar algo en el documento en el lugar donde se le quiere incluir.

👉🏻 Texto preformateado: Etiquetas "pre" y "code".

En el código HTML no se respetan los espacios en blanco ni los saltos de línea, de manera que son interpretados como un único espacio en blanco entre palabras al escribirse en la página.

Sin embargo esto podemos cambiarlo mediante la etiqueta: pre

<pre>
Lista de elementos:
  - Elemento 1
  - Elemento 2
  - Elemento 3
</pre>

Resultado:

Lista de elementos:
  - Elemento 1
  - Elemento 2
  - Elemento 3

La etiqueta code da también al texto el mismo tipo de letra (courier), sin embargo no respeta ni los espacios en blanco ni los saltos de línea. Veamos el mismo código anterior cambiando la etiqueta:

<code>
  Lista de elementos:
    - Elemento 1
    - Elemento 2
    - Elemento 3
</code>

Resultado:

Lista de elementos: - Elemento 1 - Elemento 2 - Elemento 3

Aunque a veces se suelen utilizar ambas, ya que La etiqueta code añade un matiz semántico. Indica a los navegadores que el fragmento de código preformateado indica precisamente un código, y que no debe ser interpretado como una etiqueta HTML.

<pre>
  <code>
    ...
  </code>
</pre>

👉🏻 Listas

En HTML podemos utilizar listas que permiten describir o enumerar elementos.

Las listas pueden ser de tipo:

  • ul: Lista desordenada:
    Los elementos de la lista no están ordenados y van precedidos por una viñeta (como ésta lista que estás leyendo).
  • ol: Lista ordenada:
    Los elementos de la lista van precedidos de una numeración, bien sea con números o con letras.

Dentro de cada lista, puedes utilizar cualquier elemento li para definir un elemento de lista.

Por Ejemplo:

👉🏻 Lista Ordenada:

  1. Elemento de lista
  2. Elemento de lista
  3. Elemento de lista

Podemos usar el atributo <ol start="6"></ol> para definir el orden inicial de la lista.

  1. Elemento de lista
  2. Elemento de lista
  3. Elemento de lista

Invertir el orden de la lista con el atributo <ol reversed="reversed"></ol>

  1. Elemento de lista
  2. Elemento de lista
  3. Elemento de lista

Cambiar el tipo de lista con el atributo <ol type="a"></ol>

  1. Elemento de lista
  2. Elemento de lista
  3. Elemento de lista

Sintaxis:

<ol>
  <li>Elemento de lista 1</li>
  <li>Elemento de lista 2</li>
  <li>Elemento de lista 3</li>
</ol>

👉🏻 Lista Desordenada:

  • Elemento de lista

👉🏻 Tablas

El lenguaje HTML permite la introducción de tablas en las páginas web. Para ello tiene una serie de etiquetas especiales para construir las tablas. Sin embargo, antes de nada, y para evitar confusiones, vamos a definir los elementos básicos de una tabla:

  • Tabla: Espacio rectangular dividido por líneas horizontales y verticales de manera que se forman diferentes compartimentos o casillas.
  • Fila: Cada una de las líneas horizontales de casillas de que consta la tabla.
  • Columna: Cada una de las líneas verticales de casillas de que consta la tabla.
  • Celda: Cada una de las casillas de la tabla

Etiquetas básicas:

  • table Representa datos en dos o mas dimensiones.
  • tr Define una fila de celdas en una tabla.
  • td Define la celda de una tabla que contiene datos.
  • th Define una celda como encabezado de un grupo de celdas en una tabla.

Etiquetas adicionales:

  • caption Define un título para una tabla.
  • col Define una columna de una tabla.
  • colgroup Define un grupo de columnas de una tabla.

Secciones para tablas:

  • thead Define el encabezado de una tabla.
  • tbody Define el cuerpo de una tabla.
  • tfoot Define el pie de una tabla.

Sintaxis:

<table>
  <caption>Título de la tabla</caption>
  <col>
  <colgroup>
  <thead>
    <tr>
      <th>Encabezado 1</th>
      <th>Encabezado 2</th>
      <th>Encabezado 3</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Celda 1</td>
      <td>Celda 2</td>
      <td>Celda 3</td>
    </tr>
    <tr>
      <td>Celda 1</td>
      <td>Celda 2</td>
      <td>Celda 3</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Pie 1</td>
      <td>Pie 2</td>
      <td>Pie 3</td>
    </tr>
  </tfoot>
</table>

Resultado:

Título de la tabla
Encabezado 1 Encabezado 2 Encabezado 3
Celda 1 Celda 2 Celda 3
Celda 1 Celda 2 Celda 3
Pie 1 Pie 2 Pie 3

Para más información sobre las tablas, puedes consultar la documentación en MDN Tablas HTML

👉🏻 Etiquetas de Estructura

Además de definir partes individuales de tu página (como un párrafo o una imagen), HTML también dispone de elementos de bloque que se pueden utilizar para estructurar las distintas áreas de tu sito web (tal como el encabezado, el menú de navegación o la parte del contenido principal.

  • header: Etiqueta para encabezado principal de la página.
  • footer: Etiqueta que representa un grupo de contenido al final de una página.
  • nav: Etiqueta para la navegación principal de la página
  • article: Etiqueta para un bloque de contenido que tiene sentido por sí mismo aparte del resto de la página (por ejemplo una entrada en un blog).
  • section: Etiqueta para una sección en sí misma (por ejemplo un minimapa o un conjunto de titulares y resúmenes).
  • aside: Etiqueta para contenido que no está directamente relacionado con el contenido principal, pero que puede aportar información adicional relacionada indirectamente con él (resúmenes, biografías del autor, enlaces relacionados, etc.).
  • main: Etiqueta que encierra el contenido principal de la página. Utilizar solamente una vez para cada página y lo situaremos directamente dentro del elemento body. Es mejor que no lo anidemos en otros elementos.
  • div: Etiqueta para contenido que no tiene semántica

Para más información sobre las etiquetas de estructura, puedes visitar la Estructura web y documentación
